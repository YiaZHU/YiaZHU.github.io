## 使用建议

这100篇文献覆盖了该综述应包含的**10个核心模块**：

```
一年写作规划（4月至次年3月）

第一季度   文献精读 + 框架确定
├── 模块①②：总览 + IVF临床AI（确定切入点）
└── 模块⑨⑩：方法论 + 伦理（建立论证基础）

第二季度   核心内容撰写
├── 模块③④⑤：单细胞/空间组学 + GRN（技术主线）
└── 模块⑥：表观组学（机制主线）

第三季度   进阶内容撰写
├── 模块⑦：类胚体模型（前沿应用）
└── 模块⑧：活体成像AI（技术创新）

第四季度 整合、修订、投稿
└── 全稿审改 + 图表制作 + 期刊选择
```

**目标期刊建议**（根据综述定位）：

- 偏临床：*Human Reproduction Update*（IF ~15）, *Fertility & Sterility*
- 偏技术：*Briefings in Bioinformatics*, *Nature Methods*（难度较高）
- 偏综合：*Development*, *Cell Systems*, *iScience*
